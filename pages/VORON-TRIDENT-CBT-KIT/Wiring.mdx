# Wiring Guide

import { Callout } from 'nextra/components'

## Before You Start

<Callout type="info">
  Make sure your LRS-200-24 110V/230V Switch is in the right orientation.
</Callout>

<Callout type="info">

Important:

We strongly recommend to flash M8P/SB2209 before you install them, check https://mpx-wiki.vercel.app/Firmware-Flash/flash-m8p-v1-ebb or https://mpx-wiki.vercel.app/Firmware-Flash/flash-m8p-v2-ebb  (Depending on your M8P version)

This usually saves you 2 hours or even more time.

</Callout>


### BIGTREETECH SPI35 Touch Screen

<Callout type="info">

We strongly recommend you install the FPC cable on the M8P side before you install on DIN rails.

</Callout>

Uncomment this line from BoardEnv.txt, this file is in CB1 TF card /boot

##  Layout

Use 3M 5952 Tape to mount PVC trunkings. 

<Callout type="info">

We recommend to clean the area to be bonded with alcohol or IPA (Isopropyl Alcohol). 

Our PSU mounting is different from the official one. Please refer to the Print Guide for instructions

Notice the SSR direction. 

</Callout>

We strongly recommend to install the display FPC cable before mounting the M8P. 

The connector is on the back of M8P.

## Toolboard 

Note: you need to remove the outer sheath of SB2209 cable for some length on electronic bay side.


### Toolboard 24V 

<Callout type="warning">

Note:

The default wires are tinned, solder cold flows/creeps under the load and can become a fire-hazard.

You can crimp it with a pin terminal or cut it off and use bare wire instead. 

</Callout>
